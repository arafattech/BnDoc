# Setup Guide

This guide will help you get started with the <a href="https://dev.hybri.tech/en/login" target="_blank"> MES Portal</a>

## Prerequisites

Before you begin, ensure you have the following requirements met:
- Modern web browser (Chrome, Firefox, or Edge)
- Stable internet connection
- Authorized user credentials

## Step-by-Step Configuration

1. **Access the Portal**: Navigate to the official <a href="https://dev.hybri.tech/en/login" target="_blank"> MES Login</a>.

![Login Page](/img/login.png)

2. **Login**: Use your provided employee ID and password.
3. **Environment Setup**: Follow the on-screen instructions to configure your initial dashboard.

## Support

If you encounter any issues during setup, please contact the IT support team.
